# 🦠 COVID-19 Global Data Analysis

Time Series Analysis and Visualization of COVID-19 
global data using Python. Covers cases, deaths, 
and recovery rates by country and region.

---

## 📊 Project Overview

Analyzed COVID-19 data from January 2020 to July 2020
covering 187 countries to understand the global impact
of the pandemic.

---

## 🔍 What I Did

- Loaded and cleaned 3 different datasets
- Handled missing values and fixed data types
- Analyzed global trend over time
- Found top countries by cases and death rate
- Compared WHO regions by death rate
- Identified best recovery rate countries

---

## 📈 Key Findings

- Total confirmed cases: 16,480,485
- Total deaths: 654,036
- Total recovered: 9,468,087
- USA had most cases — 4,290,259
- Yemen had highest death rate — 28.56%
- Qatar had best recovery rate — 97.02%
- Europe had worst death rate — 6.4%
- Africa had lowest death rate — 1.69%
